
> Problem: [7. 整数反转](https://leetcode.cn/problems/reverse-integer/description/)

[TOC]

# 思路

> 利用位操作对整数反转

# 解题过程

> 
- 理解整数的位操作：
对于任意正整数 x： 
• x % 10 可以提取其最后一位数字；
 • x / 10（整数除法）可以去掉最后一位，保留其余高位。   
-   反转整数：
利用上述性质，通过循环不断取出 x 的末位，并将其拼接到新变量 t 的末尾（t = t * 10 + x % 10），从而实现整数的反转。 

# 复杂度

- 时间复杂度: $O(logx)$
- 空间复杂度: $O(logx)$



# Code
```C []
#define max pow(2,31)
int reverse(int x) {
    long sum=0;
    while(x!=0)
    {
       int m=x%10;
       sum=sum*10+m;
        x=x/10;
    }
    if(sum<-max||sum>max-1) return 0;
    return sum;
}
```
