
> Problem: [1672. 最富有客户的资产总量](https://leetcode.cn/problems/richest-customer-wealth/description/)

[TOC]

# 思路

> 1、初始化一个待比较的变量weath
  2、遍历整个二维列表，累加每个子列表内部元素的和
  3、分别和weath比较，如果大于就更新weath的值，直到遍历结束
  4、返回weath


# 解题过程

> 1、for循环遍历整个二维列表
  2、if语句对weath和子列表之和做判断

# 复杂度

- 时间复杂度: $O(m*n)$
- 空间复杂度: $O(1)$



# Code
```Python3 []
class Solution:
    def maximumWealth(self, accounts: List[List[int]]) -> int:
        weath=0
        for account in accounts:
            current=sum(account)
            if current>weath:
                weath=current
        return weath
```
