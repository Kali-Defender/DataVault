
> Problem: [1437. 是否所有 1 都至少相隔 k 个元素](https://leetcode.cn/problems/check-if-all-1s-are-at-least-length-k-places-away/description/)

[TOC]

# 思路

> 用last记录上一个 1 出现的位置，每当遇到新的 1 时，检查它与上一个 1 的距离是否满足 ≥ k+1（即中间至少有 k 个 0）。

# 解题过程

> 1、定义一个极小的变量last确保第一个1不会被误判
2、for循环遍历整个数组，如果找到一个1，如果不满足条件就返回false，把i赋值给last，继续循环，直到循环结束。


# 复杂度

- 时间复杂度: $O(n)$
- 空间复杂度: $O(1)$



# Code
```C []
bool kLengthApart(int* nums, int numsSize, int k) {
    int last=-100000;
    for(int i=0;i<numsSize;i++){
        if(nums[i]==1)
        {
            if(i-last<=k){
            return false;
        }
        last=i;
    }
    }
    return true;
}
```
