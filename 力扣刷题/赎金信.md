
> Problem: [383. 赎金信](https://leetcode.cn/problems/ransom-note/description/)

[TOC]

# 思路

> 1、用 `Counter` 分别统计 `ransomNote` 和 `magazine` 里字符的出现次数，然后比较。
2、若 `ransomNote` 中每个字符的次数都不超过 `magazine` 里对应字符的次数，就返回 `True`（能构成赎金信），否则返回 `False`。

# 解题过程

# 复杂度

- 时间复杂度: $O(m+n)$
- 空间复杂度: $O(m+n)$



# Code
```Python3 []
class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
         return Counter(ransomNote) <= Counter(magazine)
```
