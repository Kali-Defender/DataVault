
> Problem: [1. 两数之和](https://leetcode.cn/problems/two-sum/description/)

[TOC]

# 思路

> 依据题意遍历两次数组，将符合条件的数组元素下标存入新数组

# 解题过程

> 1.先用循环变量i遍历整个数组，再用循环变量j紧跟在循环变量i的后面遍历i的下一位到数组末尾
2.用条件限制选择符合条件的数组下标
3.定义一个新数组作为结果数组，注意要使用malloc语句定义，将符合条件的数组下标存入
4.用*returnSize做返回值条件

# 需要注意的问题
1.题目中给的变量中，"int* returnSize"不是结果数组，是结果数组的大小
# 复杂度

- 时间复杂度: $O(n²)$
- 空间复杂度: $O(1)$



# Code
```C []
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
int* twoSum(int* nums, int numsSize, int target, int* returnSize) {
    
    for(int i=0;i<numsSize-1;i++){
        for(int j=i+1;j<numsSize;j++){
        if(nums[i]+nums[j]==target&&i!=j){
            int* result = (int*)malloc(2 * sizeof(int));
            result[0]=i;
            result[1]=j;
            *returnSize=2;
            return result;
        }
    }
    }
    *returnSize = 0;
    return NULL;
}
```
