# 思路

> 第一种解法：
  1、直接while循环对结果是否为零进行判断
  2、再对其奇偶性分类讨论

>第二种解法：
   1、奇偶性决定操作次数，奇数+2，偶数+1
   2、位运算 num>>=1,右移相当于除2

# Code
```Python3 []
class Solution:
    def numberOfSteps(self, num: int) -> int:
         count=0
         while num>0:
             if num%2==0:
                 num=num/2
             else:
                 num=num-1
             count+=1
         return count
```

class Solution:
    def numberOfSteps(self, num: int) -> int:
         count=0
         while num>0:
             count+=(num&1)+1 奇偶性决定步骤：奇数+2，偶数+1
             num>>=1          相当于num//2
         return max(count-1,0)防止num=0
