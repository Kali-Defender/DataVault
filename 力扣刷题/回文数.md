
> Problem: [9. 回文数](https://leetcode.cn/problems/palindrome-number/description/)

[TOC]

# 思路

> 将整数反转，与原来的整数比较，相等则是回文数，反之，不是

# 解题过程

>
- 理解整数的位操作：
对于任意正整数 x： 
• x % 10 可以提取其最后一位数字；
 • x / 10（整数除法）可以去掉最后一位，保留其余高位。   
-   反转整数：
利用上述性质，通过循环不断取出 x 的末位，并将其拼接到新变量 t 的末尾（t = t * 10 + x % 10），从而实现整数的反转。 
- 判断回文：
若反转后的数值 t 与原始值 x 相等，则说明该数为回文数，返回 true；否则返回 false。   
⚠️ 注意：负数不可能是回文数（如 -121 反转后为 121-，不符合数值定义），应直接返回 false。
>

# 需要注意的问题
>
- 要先用r变量存储x，循环过后x会变成0
- t变量的声明要足够大，防止溢出
- 判断之前先排除一些非回文数，即符合x<0||(x!=0&&x%10==0)的数
# 复杂度

- 时间复杂度: $O(logx)$
- 空间复杂度: $O(logx)$



# Code
```C++ []
class Solution {
public:
    bool isPalindrome(int x) {
        if(x<0||(x!=0&&x%10==0)) 
        return false;
        int r=x;
        long long t=0;
        while(x>0){
            t=t*10+x%10;
		    x/=10;
        }
        if(t==r){
            return true;
        }
        else{
            return false;
        }
    }
}
```
