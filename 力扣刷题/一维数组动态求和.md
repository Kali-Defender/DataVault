# 思路

> 第一种思路：动态增加（高效）
1、首先明确新数组和旧数组的关系，新数组第i个元素等于旧数组的第i个元素和前面所有的元素之和
2、利用一个for循环对nums数组遍历，nums[i]+=nums[i-1],只需要叠加前面的一个数就行
第二种思路：暴力解题（低效）
1、初始化一个新数组，将结果存入
2、两个循环，第一个循环遍历数组，第二个循环计算1到i的元素之和，存入新数组中


# 解题过程

> 直接一个for循环对nums数组遍历，并利用nums[i]+=nums[i-1]更新数组

# 遇到的问题

> 返回值错误，返回了list，这是python3的基本数据类型，导致报错

# 复杂度
第一种思路：
- 时间复杂度: $O(n)$
- 空间复杂度: $O(1)$

第二种思路
- 时间复杂度：$O(n^2)$
- 空间复杂度：$O(n)$


# Code
```Python3 []
class Solution:
    def runningSum(self, nums: List[int]) -> List[int]:
        for i in range(1, len(nums)):
            nums[i] += nums[i - 1]
        return nums
```
  
```C []
int* runningSum(int* nums, int numsSize, int* returnSize) {
    int sum=0;
    int i;
    
    
    for(i=0;i<numsSize;i++)
    {
        sum+=nums[i];
        nums[i]=sum;
    }
    
   *returnSize=numsSize;
   return nums;
}
```
